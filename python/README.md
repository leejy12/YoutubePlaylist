# Python Implementation

## Dependencies

Libraries used are [requests](https://requests.readthedocs.io/en/master/), [numpy](https://numpy.org/) and [matplotlib](https://matplotlib.org/).

## Usage

`python script.py <Playlist ID> <Google API Key>`
